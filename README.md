🚀 Initial release of AICodeDrop.

## Features
- Drag and drop project folders
- Generates a single structured text file containing the complete project
- Preserves folder hierarchy
- Excludes unnecessary files and folders
- Clean, developer-friendly output
- Ready to share with developers or coding assistants
- Fast and simple interface
