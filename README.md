# 🚀 AICodeDrop

**Turn any project into a single AI-ready text file in seconds.**

AICodeDrop is a fast, simple developer tool that scans your entire project, preserves its folder structure, extracts the relevant source code, and generates a single, well-structured text file. Perfect for sharing complete project context with coding assistants without the hassle of copying and pasting files manually.


**Try AICodeDrop right now in your browser:**  
👉 **[https://akbar-shaik.github.io/AICodeDrop/](https://akbar-shaik.github.io/AICodeDrop/)**

---

## ✨ Features

* **📂 Drag & Drop:** Easily load whole project folders.
* **🌳 Preserves Hierarchy:** Maintains your complete folder and file structure.
* **🧹 Smart Filtering:** Automatically ignores unnecessary files and folders (like `node_modules` or `.git`).
* **🤖 AI-Optimized:** Outputs a single text file with clear file separators for maximum AI context.
* **🔒 Privacy First:** Runs 100% locally in your browser. No code is ever uploaded to a server.
* **⚡ Lightning Fast:** Processes projects of all sizes instantly.
* **📋 Export Options:** One-click copy to clipboard or download as a file.

## 📖 How It Works

1. Open [AICodeDrop](https://akbar-shaik.github.io/AICodeDrop/).
2. Drag and drop your project folder into the interface.
3. The tool instantly scans the project structure and extracts the source code.
4. A single, organized text file is generated.
5. Copy or download the output.
6. Paste the context into your favorite AI assistant and continue coding!
